# Calculator App

A simple and modern calculator web app built using HTML, CSS, and JavaScript.

## 🔥 Features

- Basic operations (+, -, \*, /)
- Clean and modern UI
- Button click working
- Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
